# Tarkov Bot Dashboard

Web dashboard for managing the Tarkov bot, viewing analytics, and updating data.

## Features

- 📊 Real-time bot statistics
- ⚙️ Bot settings management
- 💰 Price monitoring
- 🔄 Data sync status
- 📈 Analytics & usage graphs

## Setup

1. Install dependencies: `npm install`
2. Update `API_URL` in `.env`
3. Start: `npm start`
4. Build for production: `npm build`

## Deployment

Deploy to Vercel:
```bash
vercel deploy
```

## Environment Variables

```
REACT_APP_API_URL=https://api.example.com
REACT_APP_BOT_INVITE=https://discord.com/oauth2/authorize?...
```
